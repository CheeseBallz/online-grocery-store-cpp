# Online Grocery Store (C++)

A complete console-based Online Grocery Store Management System built in C++.

## Features
- Account Creation & Login
- Password Strength Validation
- Shopping Cart System
- Loyalty Points
- Billing with Discount
- Card & COD Payment
- Shipment Tracking
- Complaint & Feedback System

## How to Run

Compile:
g++ main.cpp -o grocery

Run:
./grocery
